# Use Cases

## Smart Contract Certification
Use this repo to replay a delivered proof bundle independently of the producer's toolchain.

## Regulatory Audit
Use this repo when an auditor needs a small, inspectable verifier instead of a full theorem prover installation.

## Cross-Institutional Proof Sharing
Use this repo when one institution sends another a proof artifact and the receiver wants to replay it without trusting the sender's build environment.

## Verification Service Customers
Use this repo as the customer-side verifier for artifacts delivered by `verified-sky-checker`.
